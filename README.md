## Quick commands used
- Show IP: `ip a`
- Ping: `ping -c 5 <target>`
- Nmap quick recon: `nmap -sC -sV -oN scans/nmap_initial.txt <target>`
- Start Wireshark capture: [GUI] select `vboxnet0` and Start
